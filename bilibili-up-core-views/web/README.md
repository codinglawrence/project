# B站UP主视频核心观点提取工具 - 网页版

## 功能介绍

- 输入UP主UID，一键获取视频核心观点
- 支持DeepSeek和硅基流动两种大模型
- 响应式设计，适配各种设备
- 结果以卡片形式展示，美观易读
- 支持保存到浏览器本地和下载JSON文件
- 历史记录功能，方便查看之前的提取结果
- 可配置API密钥

## 快速开始

### 1. 安装依赖

```bash
# 回到项目根目录
cd ..

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动后端服务

```bash
# 进入web目录
cd web

# 启动Flask服务
python app.py
```

服务将在 http://localhost:5000 启动

### 3. 打开前端页面

直接在浏览器中打开 `index.html` 文件即可。

## 使用说明

### 1. 配置API密钥

- 点击页面右上角的「配置」按钮
- 在弹出的模态框中输入你的API密钥
- 点击「保存配置」

### 2. 提取核心观点

1. 在「UP主UID」输入框中输入目标UP主的UID
2. 设置要处理的最大视频数（默认10）
3. 选择要使用的模型（DeepSeek或硅基流动）
4. 点击「开始提取」按钮
5. 等待处理完成，查看提取结果

### 3. 查看结果

- 结果将以卡片形式展示，包含视频标题、发布时间、视频链接和核心观点
- 每个结果卡片可以点击「查看视频」直接跳转到B站视频页面

### 4. 保存结果

- 点击「保存到本地」按钮，可以将结果保存到浏览器本地存储
- 同时会自动下载一个JSON格式的结果文件

### 5. 历史记录

- 所有提取结果都会自动保存到历史记录中
- 可以在历史记录中查看之前的提取结果
- 可以删除不需要的历史记录

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **后端**: Python + Flask
- **API**: Bilibili API + 大模型API（DeepSeek/硅基流动）
- **样式**: 自定义CSS，使用Font Awesome图标
- **存储**: 浏览器本地存储（localStorage）

## 注意事项

1. 确保后端服务已经启动
2. 确保API密钥配置正确
3. 处理大量视频可能需要较长时间
4. 浏览器本地存储有容量限制，建议定期清理历史记录
5. 如遇到跨域问题，请检查浏览器控制台的错误信息
